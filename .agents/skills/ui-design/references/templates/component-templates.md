# Component Templates — 组件模板

本文档定义常用组件的规范模板。

---

## Component 1: Button

### Primary Button

```
┌─────────────────┐
│                 │  ← 默认
│     确认        │
└─────────────────┘

┌─────────────────┐
│                 │  ← 按压 (scale: 0.95)
│     确认        │
└─────────────────┘

┌─────────────────┐
│                 │  ← 禁用 (opacity: 0.5)
│     确认        │
└─────────────────┘

┌─────────────────┐
│   ⏳ 处理中...  │  ← 加载中 (spinner + 文字变化)
└─────────────────┘
```

### Secondary Button

```
┌─────────────────┐
│                 │
│     取消        │  ← 默认 (边框样式，无填充)
└─────────────────┘
```

### Ghost Button

```
    取消           ← 无边框无背景，纯文字 + hover 背景
```

### Danger Button

```
┌─────────────────┐
│                 │
│     删除        │  ← 红色主题
└─────────────────┘
```

## Component 3: Checkin Button（大号打卡按钮）

### 未打卡状态

```
        ┌─────────────────┐
        │                 │
        │                 │
        │       ✓         │  ← 主色背景，中心勾
        │                 │
        │                 │
        └─────────────────┘
             点击打卡
```

### 按压状态

```
        ┌─────────────────┐
        │                 │
        │                 │
        │       ✓         │  ← 缩放 0.95 动画
        │                 │
        │                 │
        └─────────────────┘
             打卡中...
```

### 已打卡状态（成功动画）

```
        ┌─────────────────┐
        │                 │
        │                 │
        │       ✓         │  ← 弹跳 + 放大动画
        │                 │
        │                 │
        └─────────────────┘
           🔥 连续 15 天
```

### 打卡成功庆祝动画序列

```
Step 1: 点击            Step 2: 弹跳           Step 3: 粒子扩散
┌───────┐             ┌───────┐              ┌───────┐
│   ✓   │      →     │  ✓✨  │       →      │ ✓ ✨  │
└───────┘             └───────┘              └───────┘
                            ↓                    ↓
                       ╱ ✨ ╲              ✱  ✨  ✱
                      ╱ ✨  ╲             ✱  ✨  ✱
                         ✨               ✱  ✨  ✱
```

## Component 5: Input Fields

### 文本输入

```
┌─────────────────────────────┐
│ 习惯名称                    │  ← 标签
│ ┌─────────────────────────┐│
│ │ 输入习惯名称...           ││  ← 输入框
│ └─────────────────────────┘│
│                            │
│ 名称不能为空                │  ← 错误提示
└─────────────────────────────┘

聚焦状态：
┌─────────────────────────────┐
│ 习惯名称                    │
│ ┌─────────────────────────┐│
│ │ 输入习惯名称...           ││  ← 边框变主色
│ └─────────────────────────┘│
└─────────────────────────────┘
```

### Toggle 开关

```
提醒通知    [ON ]     ← 开启
提醒声音    [OFF]     ← 关闭
```
