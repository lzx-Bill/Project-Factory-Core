# Page Templates

以下模板只提供布局结构。使用时必须替换为当前项目的术语、字段和操作。

## List / Collection

```text
┌──────────────────────────────────┐
│ 页面标题              [主要操作] │
├──────────────────────────────────┤
│ [搜索] [筛选] [排序]             │
├──────────────────────────────────┤
│ 条目标题        状态        >    │
│ 摘要信息                         │
├──────────────────────────────────┤
│ 条目标题        状态        >    │
│ 摘要信息                         │
└──────────────────────────────────┘
```

必须补充：loading、empty、error、分页/继续加载、批量操作（如确有需求）。

## Detail

```text
┌──────────────────────────────────┐
│ ← 返回   资源标题      [操作菜单] │
├──────────────────────────────────┤
│ 状态与关键摘要                   │
├──────────────────────────────────┤
│ 主要内容                         │
│                                  │
├──────────────────────────────────┤
│ 关联信息 / 操作记录              │
└──────────────────────────────────┘
```

必须补充：不存在、无权限、删除/归档确认、并发更新冲突（如适用）。

## Create / Edit Form

```text
┌──────────────────────────────────┐
│ ← 返回   创建或编辑              │
├──────────────────────────────────┤
│ 字段标签                         │
│ [输入控件                    ]    │
│ 帮助或错误文本                   │
│                                  │
│ 字段标签                         │
│ [输入控件                    ]    │
├──────────────────────────────────┤
│ [取消]                    [保存] │
└──────────────────────────────────┘
```

必须补充：字段校验、提交中、提交失败、离开未保存页面、成功后的去向。

## Dashboard

```text
┌──────────────────────────────────┐
│ 页面标题            [时间/范围]  │
├──────────┬──────────┬────────────┤
│ 指标 A   │ 指标 B   │ 指标 C     │
├──────────┴──────────┴────────────┤
│ 趋势或分布                       │
├──────────────────────────────────┤
│ 待处理事项 / 最近活动            │
└──────────────────────────────────┘
```

无真实指标需求时不要创建 Dashboard。

## State Template

```text
┌──────────────────────────────────┐
│ [图标或进度]                     │
│ 状态标题                         │
│ 原因与下一步说明                 │
│ [恢复动作]                       │
└──────────────────────────────────┘
```

用于 loading、empty、error、offline、no-permission。恢复动作必须真实可用。
