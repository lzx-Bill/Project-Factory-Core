# Page Templates — 页面模板

本文档定义常用页面模板的 ASCII 线框图规范。

---

## Template 1: List View（列表视图）

**适用场景**: 习惯列表、打卡记录、任务列表

```
┌─────────────────────────────────┐
│  ← 返回     页面标题        ⚙️ │  ← 顶部栏（可选返回/菜单）
├─────────────────────────────────┤
│  [搜索框]                       │  ← 搜索区（可选）
├─────────────────────────────────┤
│                                 │
│  ┌─────────────────────────┐   │
│  │ ● 习惯名称         🔥15 │   │  ← 列表项（圆点=状态）
│  │   今日: ✓ 已完成        │   │
│  └─────────────────────────┘   │
│                                 │
│  ┌─────────────────────────┐   │
│  │ ○ 习惯名称              │   │  ← 未完成状态
│  │   今日: 未打卡          │   │
│  └─────────────────────────┘   │
│                                 │
│  ┌─────────────────────────┐   │
│  │ ○ 习惯名称         ⚠️中断 │   │
│  │   已连续 3 天，昨日中断  │   │
│  └─────────────────────────┘   │
│                                 │
├─────────────────────────────────┤
│  🏠 首页   📊 统计   ➕ 打卡   │  ← 底部 Tab Bar
└─────────────────────────────────┘
```

### 变体：带滑动操作

```
┌─────────────────────────────────┐
│  ← 返回     习惯列表        ⚙️ │
├─────────────────────────────────┤
│                                 │
│  ┌─────────────────────────┐←→ │
│  │ ● 每天运动30分钟   🔥15 │ ⚙️│  ← 左滑显示编辑
│  │   ✓ 今日已完成          │ 🗑️│  ← 右滑显示删除
│  └─────────────────────────┘   │
│                                 │
│  ┌─────────────────────────┐←→ │
│  │ ○ 每天读书           🔥8 │ ⚙️│
│  │   今日: 未打卡          │ 🗑️│
│  └─────────────────────────┘   │
│                                 │
└─────────────────────────────────┘
```
