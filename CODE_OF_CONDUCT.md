# Community Code of Conduct

Project Factory Core 希望建立一个尊重、务实、可验证的开源协作环境。本准则适用于 Issue、Pull Request、Review、Discussion 以及代表本项目进行的公开交流。

## 我们鼓励

- 尊重不同经验、背景和观点
- 围绕问题、证据和可执行改进展开讨论
- 清楚区分事实、假设和个人偏好
- 善意给出并接受建设性反馈
- 对自己的言行和贡献负责，发现问题及时修正

## 不可接受的行为

- 骚扰、威胁、歧视、侮辱或人身攻击
- 未经允许公开他人的私人信息
- 持续打断讨论、恶意挑衅或针对个人施压
- 冒充他人、逃避处置或隐瞒内容来源
- 与项目协作无关的垃圾信息和重复推广
- 其他会破坏安全、尊重和有效协作的行为

## 处理方式

维护者可以编辑、隐藏或删除违反本准则的内容，并根据影响采取提醒、临时限制或永久禁止参与等措施。处理时优先保护报告人的隐私，并尽量记录一致、可解释的判断依据。

## 报告问题

请通过维护者 [@lzx-Bill](https://github.com/lzx-Bill) 的 GitHub 主页请求私密联系方式。公开消息中只说明需要报告社区行为问题，不要附带当事人信息、截图或其他敏感细节。

一般技术问题请使用仓库 Issue 模板；安全漏洞请遵循 [SECURITY.md](SECURITY.md)。
